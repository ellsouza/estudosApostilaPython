# Apostila Interativa de Python

Material de estudo criado para revisar conceitos fundamentais de Python, organizado a partir dos meus próprios arquivos de prática do curso.

🔗 **Acesse a apostila online:**  
https://ellsouza.github.io/estudosApostilaPython/

## Sobre o projeto

Este repositório reúne meus estudos iniciais em Python em formato de apostila interativa. A ideia foi transformar arquivos `.py` feitos durante o curso em um material mais organizado, visual e fácil de consultar sempre que eu quiser revisar algum conceito.

Além de praticar Python, usei este projeto para exercitar Engenharia de Prompt e o uso de IA como apoio ao aprendizado, conectando meus estudos em programação com meu foco atual em IA, LLMs e Machine Learning.

## Como rodar localmente

Clone o repositório:

```bash
git clone https://github.com/ellsouza/estudosApostilaPython.git
cd estudosApostilaPython
```

Abra o arquivo `index.html` no navegador para ver a apostila.

Para rodar os arquivos Python:

```bash
python conceitosBasicosPython.py
python listas.py
python dicionario.py
python conjuntos.py
python funcoes.py
python errosComunsPython.py
python entradasSaidas.py
python leituraEscritaArquivos.py
python importacaoCriacaoModulos.py
```

Para os exemplos com Flask:

```bash
pip install -r requirements.txt
python tabuada.py
```

Depois, abra no navegador o endereço indicado no terminal, geralmente:

```text
http://127.0.0.1:5000
```
